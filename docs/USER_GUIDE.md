# GATEMATE User Documentation

## Getting Started

### Installation

#### Mobile App
1. Download **Expo Go** from App Store or Google Play
2. Scan the QR code from `npx expo start`
3. Login with your credentials

#### Web App (PWA)
1. Open https://gatemate.example.com in your browser
2. Click "Install" when prompted to add to home screen
3. Login with your credentials

## Gate Control

Tap on a gate to access detailed controls:

### Position Slider
- Drag slider to set exact opening percentage (0-100%)
- Presets: 25%, 50%, 75%, Full Open

### 3D Visualization
- Shows real-time gate position
- Animates during movement

### Stop Button
- Red stop button for emergency halt
- Gate will stop at current position

---

## Schedules

Automate gate operations:

1. Tap **+ Add Schedule**
2. Set time using the time picker
3. Choose action: **Open** or **Close**
4. Select recurrence: Daily, Weekdays, Weekends, Custom
5. Tap **Save**

### Managing Schedules
- Toggle switch to enable/disable
- Swipe left to delete
- Tap to edit

## Settings

### Device Settings
- **Device Name** - Rename your gate
- **Network** - WiFi connection status
- **Firmware** - Current version and updates

### Security
- **Require PIN** - Ask for PIN before controls
- **Auto-Lock** - Lock app after 5 minutes
- **Security Audit** - View access history

### Notifications
- **Push Notifications** - Gate events
- **Emergency Alerts** - Obstacle detection, power outages
- **Email Reports** - Weekly activity summary

## Troubleshooting

### Gate Not Responding
1. Check if device shows "Online" status
2. Verify WiFi connection
3. Try the Emergency Stop button
4. Power cycle the gate controller

### Connection Issues
1. Check your internet connection
2. Log out and log back in
3. Restart the app

### Obstacle Detected
1. Clear any obstructions from gate path
2. Press "Dismiss" on the warning
3. Retry the operation
